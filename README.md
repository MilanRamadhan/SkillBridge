# 🌉 SkillBridge — AI-Driven Career Path & Gap Analysis

SkillBridge adalah platform berbasis kecerdasan buatan (AI) yang dirancang untuk membantu menjembatani kesenjangan keahlian (*skill gap*) antara profil lulusan/pencari kerja dengan kebutuhan nyata di industri dunia kerja. 

Proyek ini dikembangkan sebagai **Capstone Project** untuk program **Pijak**.

---

## 🚀 Fitur Utama

- **AI Resume Parser**: Mengekstrak informasi skill, pengalaman kerja, dan edukasi dari berkas CV (PDF/DOCX) secara instan menggunakan NLP.
- **Skill Gap Analysis**: Membandingkan keahlian Anda saat ini dengan standar industri kompetensi yang dituju secara akurat.
- **Personalized Learning Path**: Menyusun rekomendasi modul atau peta jalan pembelajaran terarah guna menutup celah keahlian Anda dengan efisien.
- **Interactive Team Profile**: Halaman khusus yang memperkenalkan anggota kelompok di balik layar pembuatan SkillBridge.
